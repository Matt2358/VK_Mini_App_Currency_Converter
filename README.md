Конвертер валют – простое и удобное приложение для пересчёта сумм из одной валюты в другую.
Как пользоваться:
Откройте приложение в VK (веб-версия или мобильное приложение).
В поле «Количество» введите сумму, которую хотите конвертировать (по умолчанию 100).
В первом селекте «ИЗ» выберите исходную валюту (например, RUB).
Во втором селекте «В» выберите целевую валюту (например, USD).
Результат конвертации отобразится сразу под селектами. Также будет показан обратный курс (например, 1 RUB = X USD).
Чтобы быстро поменять валюты местами, нажмите кнопку с двумя стрелками ⇅.
Если вы часто используете определённую пару, нажмите на иконку звёздочки рядом с курсом – пара сохранится в блоке «Избранные пары» внизу.
Для переключения на сохранённую пару просто кликните по ней в списке – поля автоматически заполнятся.
Чтобы удалить пару из избранного, нажмите крестик справа от неё.
Курсы валют обновляются автоматически каждый час. Вы также можете обновить их вручную кнопкой «Обновить курсы». При обновлении кнопка становится неактивной и отображается индикатор загрузки.
Если интернет-соединение отсутствует, приложение покажет сообщение об ошибке, но продолжит работу с последними загруженными (или запасными) курсами.
Приложение адаптировано для всех типов устройств и поддерживает тёмную тему VK.
